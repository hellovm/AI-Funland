# AI Funland

A local AI Q&A interaction platform powered by OpenVINO.

- Current version: V0.0.1 Dev
- Date: 2025-11-19

## Features

- Model selection and chat
- Model management: download, quantize, delete
- System hardware info and accelerator selection: CPU, Intel GPU, Intel NPU, NVIDIA GPU
- Bilingual UI (English/中文)
- Responsive, modern UI

## Quick Start (Windows)

1. Double-click `start.bat`
2. The script downloads Python embedded, installs dependencies, starts backend, opens browser

## Dependencies

- apiflask==2.4.0
- openvino==2025.3.0
- openvino-genai==2025.3.0.0
- openvino-tokenizers==2025.3.0.0
- langchain_community==0.3.29
- optimum[openvino]==1.27.0
