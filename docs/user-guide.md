# 用户操作文档 / User Guide

## 下载进度优化
- 在模型管理面板输入 ModelScope 模型ID，点击“下载”。
- 进度条实时显示百分比、速度（KB/s 或 MB/s）、剩余时间（ETA）。
- 可随时点击“Pause/Resume”进行暂停/继续（在单个文件下载间隙生效）。
- 错误会在进度区域显示，并记录到日志。

## 量化参数配置
- 在“量化参数配置”面板选择：
  - 精度：8 或 4 位
  - 算法：权重量化、Hybrid、Mixed（可选 NF4 + F8E4M3）
  - 范围：Per-tensor 或 Per-channel
  - 权重数据类型：int8 / nf4 / int4
  - 激活数据类型：f8e4m3 / int8
  - 数据集：可选（用于某些混合或激活量化场景）
- 参数验证：不合法参数会提示错误。
- 预设管理：可保存、加载、导出、导入 JSON 预设。
- 点击“量化”按钮执行量化并在进度区查看状态。

## 界面与适配
- 响应式布局，窄屏自动切为单列显示。
- 所有操作出现错误会提示消息。

## 日志
- 后端将关键操作写入 `data/logs/app.log`（若存在）。

## 常见问题
- 下载速度低：检查网络或更换模型源；下载暂停在当前文件完成后生效。
- 量化失败：确认模型ID与参数组合有效；必要时提供 `dataset`。
